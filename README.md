# ETL pipeline for recon jobs
 - Apache camel based ETL pipeline for recon jobs
 - Pipeline can be used for batch and stream processing
 - Built it observability features
 - Pipeline can be started and stopped using REST API
- Auto start and stop of pipelines based on configuration or database

## Pipeline Creation

Sample: 
 - com.company.etl.pipeline.reconone.pipelines.pipeline1 (Batch)
 - com.company.etl.pipeline.reconone.pipelines.pipeline2 (Stream)
